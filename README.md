#  Takeloca
写真日記投稿サービスです。都道府県別に記事を検索することもできます。

## アプリ概要
写真を日記のように投稿できるサービスです。また、投稿時には撮影場所を登録することができます。
投稿した記事は投稿一覧から都道府県別に検索することができます。記事詳細には都道府県だけでなく、詳細な撮影場所が記載されています。

##　機能
Ruby on Rails で作成したアプリケーションです。
- ログインしていなくても記事一覧は閲覧可能であるが、記事の投稿、検索機能、お気に入り機能は使用できないように実装
- レスポンシブ対応

##　使い方
1. 右上の新規登録よりユーザ登録
2. 投稿一覧より、投稿一覧や検索機能を使い見たい記事を検索する
3. Postにて記事の投稿や記事詳細からお気に入りをすることができる
